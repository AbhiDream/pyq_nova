A particle moves along a straight line such that its displacement at any time t is given by s = (t³ - 6t² + 3t + 4) meters. The velocity when the acceleration is zero is:

lightbulb Solution

Given: s = t³ - 6t² + 3t + 4
Velocity v = ds/dt = 3t² - 12t + 3
Acceleration a = dv/dt = 6t - 12
When acceleration is zero: 6t - 12 = 0 => t = 2s
Substituting t = 2 in velocity equation:
v = 3(2)² - 12(2) + 3 = 12 - 24 + 3 = -9 m/s
